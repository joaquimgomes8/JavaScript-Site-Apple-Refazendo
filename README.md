# 🍎 Apple Site – Recriação

Este projeto é uma **recriação do site da Apple**, desenvolvida com o objetivo de **praticar e aprimorar habilidades em desenvolvimento web**, focando em estrutura e layout.

---

## 🚀 Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript
- Git e GitHub
